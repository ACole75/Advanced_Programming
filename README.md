README

Code Generator Instructions


The program is designed to take a user input (standard FDS). It will then parse for relevant information and filter out data which is not required, which will then format into the correct studio 5000 ASCII format code that can be pasted into a ladder file. The program uses Python but is packaged as an exe so that you are not required to have any prerequisite installations on
your machine. 

In order to run the program you simply open the exe file which will execute the code, then you are required to enter the input file, and output directory, along with a sequence name
(required). The program will then run, extract the data, format it and output this into a text file ready to paste into your Studio 5000 ladder.

STEPS

1. Open the exe

2. Select an input file (this will be your FDS), simply search for its location and double click it

3. Select an output directory, this is where your text file will be outputted to

4. Enter a sequence name, this is simply the sequence number and is used to better search the document

5. Click convert, this will run the main code and output the converted text 

6. Copy the text directly from the generated text file and paste into the desired ladder file in Studio 5000
